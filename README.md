# MeshLab Site

This site is generated with the [hugoplate](https://github.com/zeon-studio/hugoplate) theme with the [Hugo static site generator](https://gohugo.io/).

## Dependencies

- Hugo version v0.153.2 or newer
- Node v22 or newer
- Go v1.24 or newer

## Run locally

```
npm install
npm run dev
```

## Deploy

```
npm run build
```
